# ai-career-diagnosis セットアップ記録

## 2026-06-11 実施内容

### インストールしたパッケージ（15個）

**デザイン系**
- shadcn/ui（Tailwind v4対応・初期化済み）
- lucide-react（アイコン）
- framer-motion（アニメーション）
- clsx（クラス名結合）
- tailwind-merge（Tailwindクラスの競合解決）
- class-variance-authority（コンポーネントのバリアント管理）

**集客・計測系**
- @vercel/analytics（ページビュー計測）
- @vercel/speed-insights（Core Web Vitals）
- next-sitemap（サイトマップ自動生成）

**マネタイズ・決済系**
- stripe（サーバーSDK）
- @stripe/stripe-js（クライアントSDK）
- @stripe/react-stripe-js（ReactコンポーネントSDK）
- @supabase/supabase-js（Webhook後のis_premium更新用）

**フォーム・バリデーション系**
- zod
- react-hook-form
- @hookform/resolvers

### 生成済み shadcn コンポーネント（`components/ui/`）
button / card / badge / progress / dialog / sheet / separator / input / label / tabs / sonner

### その他セットアップ済み
- `layout.tsx` に Analytics / SpeedInsights / Toaster 組み込み済み
- `next-sitemap.config.js` 作成・`postbuild` スクリプト追加済み
- Stripe APIルート（`app/api/checkout/` と `app/api/webhook/`）を ai-sermon-web から移植済み
- `app/success/` と `app/cancel/` ページ作成済み
- `.env.local.example` 作成済み
- `scripts/setup-stripe.mjs` で Stripe 商品・価格を自動作成済み（`STRIPE_PRICE_ID` は `.env.local` に書き込み済み）
